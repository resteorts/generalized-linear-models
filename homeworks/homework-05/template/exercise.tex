The Pareto distribution is frequently used in economics, finance, and risk modeling to describe heavy–tailed phenomena such as income, wealth, and insurance losses.

Suppose $Y_1,\dots,Y_n$ are independent observations from a Pareto distribution with density

\[
f(y \mid \alpha, y_m)
=
\frac{\alpha y_m^\alpha}{y^{\alpha+1}},
\qquad y \ge y_m,
\]

where $\alpha>0$ is the shape parameter and $y_m>0$ is a known scale parameter.

\begin{enumerate}[label=(\alph*)]

\item Rewrite the Pareto density in exponential form. Show that

\[
f(y \mid \alpha)
=
\exp
\left\{
\log \alpha + \alpha \log y_m - (\alpha+1)\log y
\right\}.
\]

\item Show that the density can be written in the exponential family form

\[
f(y \mid \theta)
=
h(y)
\exp
\left\{
\theta T(y) - A(\theta)
\right\}.
\]

Identify explicitly

\begin{itemize}
\item the natural parameter $\theta$
\item the sufficient statistic $T(y)$
\item the log-partition function $A(\theta)$
\item the base measure $h(y)$
\end{itemize}

\item Using your result from part (b), derive the likelihood for $n$ observations and identify the sufficient statistic for $\alpha$.

\item Derive the maximum likelihood estimator (MLE) for $\alpha$.



\end{enumerate}

\section*{Solution}

\subsection*{(a)}

Start with the density

\[
f(y \mid \alpha,y_m)
=
\frac{\alpha y_m^\alpha}{y^{\alpha+1}}.
\]

Rewrite using logarithms:

\[
f(y \mid \alpha)
=
\alpha y_m^\alpha y^{-(\alpha+1)}.
\]

Taking logs inside an exponential,

\[
f(y \mid \alpha)
=
\exp
\left(
\log \alpha + \alpha \log y_m - (\alpha+1)\log y
\right).
\]

\subsection*{(b)}

Rewrite the density as

\[
f(y \mid \alpha)
=
\exp
\left(
\log \alpha + \alpha \log y_m - \alpha \log y - \log y
\right).
\]

Group terms involving $\alpha$:

\[
f(y \mid \alpha)
=
\exp
\left(
\alpha(\log y_m - \log y) + \log \alpha - \log y
\right).
\]

Thus we can write

\[
f(y \mid \theta)
=
h(y)\exp\{\theta T(y) - A(\theta)\}
\]

with

\[
T(y) = -\log y
\]

\[
\theta = \alpha
\]

\[
A(\theta) = -\theta \log y_m - \log \theta
\]

\[
h(y) = \frac{1}{y}
\]

for $y \ge y_m$.

\subsection*{(c)}

For independent observations,

\[
L(\alpha)
=
\prod_{i=1}^n
\frac{\alpha y_m^\alpha}{y_i^{\alpha+1}}.
\]

Taking logs,

\[
\ell(\alpha)
=
n\log\alpha
+
n\alpha\log y_m
-
(\alpha+1)\sum_{i=1}^n \log y_i.
\]

Thus the sufficient statistic is

\[
T = \sum_{i=1}^n \log y_i.
\]

\subsection*{(d)}

Differentiate the log-likelihood:

\[
\frac{d\ell}{d\alpha}
=
\frac{n}{\alpha}
+
n\log y_m
-
\sum_{i=1}^n \log y_i.
\]

Set equal to zero:

\[
\frac{n}{\alpha}
+
n\log y_m
-
\sum \log y_i
=
0.
\]

Solve for $\alpha$:

\[
\frac{n}{\alpha}
=
\sum_{i=1}^n \log y_i - n\log y_m
\]

\[
\hat{\alpha}
=
\frac{n}{\sum_{i=1}^n \log(y_i/y_m)}.
\]
